# Installation

## Via pip (recommended)

```bash
pip install sentinel-security
```

## Using Docker

```bash
docker run ghcr.io/ronaldgosso/sentinel:latest scan .
```

## Using the GitHub Action

See the [CI/CD guide](ci-cd.md).

## From source

```bash
git clone https://github.com/ronaldgosso/sentinel
cd sentinel
pip install -e .
```
